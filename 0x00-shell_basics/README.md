Contains an executable pwd script that prints the absolute path name of the current working directory, executable ls script that display the contents list of the current directory, executable cd ~ script that changes the working directory to the user's home directory, executable ls -l script that display current directory contents in a long format, executable ls -la script that display current directory contents, including hidden files starting with ., executable ls -lan that display current directory contents in long format as well as the user and group IDs displayed numerically and hidden files starting with .
